Auth — lấy JWT

curl -s -X POST http://localhost:3000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"student@example.com","password":"123456"}'

Lấy result.accessToken (hoặc cookie) cho các request có Authorization: Bearer ….

Submit — chấm bài

POST /submissions (hiện public để demo; production nên bảo vệ JWT).

AI Hint — sau khi submit sai

  1. Submit code sai (xem #wrong-code), đợi status ≠ Accepted.
  2. Gọi hint (cần JWT học viên):
curl -s -X POST http://localhost:3000/problems/seed-problem-algo/hint \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -d '{"submissionId": "SUBMISSION_UUID_FROM_STEP_1"}'
Cần OPENAI_API_KEY hoặc GOOGLE_GENERATIVE_AI_API_KEY trong apps/core-api/.env. Chi tiết: AI-HINT-ASSISTANT.html.

def is_prime(x):
    if x < 2:
        return False
    for i in range(2, int(x ** 0.5) + 1):
        if x % i == 0:
            return False
    return True

print("YES" if is_prime(n) else "NO")
