AI Hint Assistant (học viên)

Socratic hints Redis rate limit Cập nhật: 2026-05-19

Gợi ý theo phong cách Socratic sau khi Run hoặc Submit thất bại trên màn /problem/[id]. Không chat tự do; không lộ đáp án hay code giải hoàn chỉnh.

Luồng

  1. Học viên Run/Submit → worker chấm → Socket submission:finished / submission:failed.
  2. Nếu status ≠ Accepted, nút nổi góc dưới phải Bạn cần trợ giúp không ^^ hiện lên — không gọi AI tự động.
  3. Học viên bấm nút → POST /problems/:problemId/hint → drawer trượt từ bên phải hiển thị gợi ý.
Contest: gợi ý AI bị tắt (submission có contestId hoặc URL ?contestId=). FAB không hiển thị.

API

  • POST /problems/:problemId/hint — yêu cầu JWT (cookie accessToken)
  • Body: { "submissionId": "uuid" }
  • Response (trong envelope chuẩn): { submissionId, hints, provider, model }

Cấu trúc hints:

{
  "summary": "…",
  "approachHints": ["…"],
  "syntaxNotes": [{ "area": "…", "note": "…" }],
  "examplePatterns": [{ "title": "…", "genericExample": "…" }],
  "encouragement": "…"
}

Mã nguồn: apps/core-api/src/ai-hint/, frontend: AiHintFab.tsx, AiHintDrawer.tsx.

Biến môi trường (core-api)

Biến Mặc định Mô tả
AI_HINT_ENABLED true Tắt toàn hệ thống: false
AI_HINT_MAX_PER_PROBLEM_PER_HOUR 10 Rate limit Redis — key ai-hint:{userId}:{problemId}
AI_HINT_DISABLED_IN_CONTEST true Mặc định chặn gợi ý khi submission có contestId. Đặt false chỉ để dev/test.
OPENAI_API_KEY / GOOGLE_GENERATIVE_AI_API_KEY — Provider LLM (cùng stack với AI testcase); thêm AI_DEFAULT_PROVIDER, model keys

Xem thêm apps/core-api/.env.example và apps/core-api/src/common/config/env.keys.ts.

Rate limit

Dùng Redis (không bảng Postgres). Mỗi lần gọi hint thành công tăng counter; cửa sổ 1 giờ (TTL 3600s).

Nếu Redis lỗi, API vẫn trả hint (fail-open) và ghi log cảnh báo — tránh chặn học viên khi hạ tầng Redis tạm lỗi.

Bảo mật

  • Chỉ chủ submission (submission.userId) được gợi ý.
  • Không gửi expected output (hidden hoặc public fail) vào LLM.
  • Post-filter redact nếu hint chứa output công khai hoặc block code quá dài.
  • Từ chối hint khi submission đã Accepted.

Postman: Request AI hint (after failed run) trong collection Core API.
